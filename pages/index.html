About me

I am a Swiss wolf named PhirosWolf, I am self-taught and I am a CS student at ETML. I really like programming, maths, everything that touches to logic, art and music. I am also a Rust enthusiast. Data privacy and the principles stated in the Mozilla Manifesto matter to me. I speak French, English and I am currently learning Russian.

I worked with many programming languages over the years and learning new ones is not a problem to me. Anyway, here is a list of the languages I learned.

Rust

Rust is one of my favorite languages. It is low-level and the syntax is more understandable than the C syntax. It is also fast, memory-safe and thread-safe. You can do whatever you want from Web applications via WebAssembly to Embedded programs and networking services.

Mastery: I know how to use advanced Rust, but I do not know all about Rust (such as how to do embedded programs).

JavaScript

JavaScript is also one of my favorite languages. It is one of my first programming language! I did a lot of things with it: Games, animations, tools for HTML Canvas and multiple other things.

Mastery: I know a lot about JavaScript. Classes, prototypes, arrow function expressions, generator functions, promise objects, async/await keywords and some JavaScript APIs.

HTML5

Even though HTML5 is not a programming language, it still fits into this list. HTML5 is a markup language. The "5" in HTML5 refers to the last version of HTML, which is a living standard.

Mastery: I know almost all HTML5 tags and how to use them.

CSS3

CSS3 is not a programming language either but still belongs to this list. CSS3 is a style sheet language. CSS3 is the most recent version of CSS.

Mastery: I know how to use CSS animations, 3D animations, CSS grids, flexboxes, pseudo-classes, media queries, etc.

PHP

PHP is a server-side language. I do not really like the style of PHP, but it works fine for creating web applications.

Mastery: I learned how to do OOP with PHP as well as how to manipulate databases with it. But I can learn more if needed.

C#

C# is a language created my Microsoft. It is pretty useful when developping applications for Windows. You can create applications for Linux, game consoles, etc. with C# but compiling C# on Linux can be really tough depending on the distribution you use.

Mastery: I know how to doo OOP with it and how to do basic networking (UDP).

Node.js

Node.js is a server-side language, like PHP, but Node.js is based on the V8 JavaScript engine. It is not used as much as PHP though. But Node.js allows you to modify the behaviour of the server itself. That is why Node.js is used by many softwares to create local servers.

Mastery: I forgot pretty much about it. But seing a bit of the documentation could remind me some basic Node.js.

Java

Java is one of my first programming languages. I used to use it for Minecraft plugins, but then, I started learning the real basics of Java.

Mastery: It is been a while since I stopped learning Java. But if I could take a look at the documentation, I could resume my learning of Java.

Swift

Swift is a programming language developed by Apple. At first glance, it may look pretty strange, but I think its syntax is better than other languages.

Mastery: I started to learn it a few months ago. I know how to do basic OOP with Swift but I still have to learn a lot about Swift.

C++

C++ is an old but reliable language. Nevertheless, the syntax is pretty hard to understand and memorise. But C++ is still used in low-level infrastructures.

Mastery: I used to do basic command-line applications with it. I even tried to use SFML. But now, I forgot almost all about it since C++ was my very first programming language. Still, C++ made me understand how pointers work as well as the basics of OOP.

SQL

SQL is not a programming language. But it is very useful when using databases. SQL can be splitted into several parts. DQL (Data Query Language), DDL (Data Definition Language), DCL (Data Control Language), DML (Data Manipulation Language) and TCL (Transaction Control Language).

Mastery: I know MySQL, but learning other "versions" of SQL should not be a problem. I know how to use DQL, DDL, DCL and DML parts of SQL.

PowerShell

PowerShell is a scripting language for Windows built on the .NET framework.

Mastery: I know the basics of PowerShell and how to manipulate databases with it.
